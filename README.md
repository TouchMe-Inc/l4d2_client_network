# About l4d2_client_network
Automatically corrects network settings

## ConVars
| ConVar                 | Value                   | Description                                 |
|------------------------|-------------------------|---------------------------------------------|
| sm_cn_only_on_connect  | 1                       | Apply settings only when client connects    |
| sm_cn_title            | "Update network settings" | Title of the MOTD window                    |
| sm_cn_message          | ""                      | Message displayed in the MOTD window        |
| sm_cn_cl_interp        | 0.0                     | Client interpolation value                  |
| sm_cn_rate             | 100000                  | Client rate setting                         |
| sm_cn_cl_cmdrate       | 100                     | Client command rate                         |
| sm_cn_cl_updaterate    | 100                     | Client update rate                          |
